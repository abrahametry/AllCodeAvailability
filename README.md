**DMCH-PneumoX#Classification.ipynb**
This script implements a deep learning pipeline for the multi-class classification of chest radiographs. It parses XML annotation files to extract labels, employing a clinical severity-based priority mapping (e.g., Pneumonia with complications > Pneumonia > Others > Normal) to resolve multi-label instances into a definitive ground truth class. The code manages data preprocessing, including calculating class distributions, executing an 80/20 train-validation split, and applying a specific fix to tolerate corrupted or truncated JPEG files. For the core modeling, it initializes and fine-tunes a pre-trained DenseNet121 neural network architecture using PyTorch. It concludes by evaluating the model and generating comprehensive validation metrics—such as precision, recall, F1-scores, and a classification report—which helps assess performance against the dataset's natural class imbalance.

**DMCH-PneumoX#Bounding Box View.ipynb**
This visualization script is designed to render and visually inspect region-level annotations within the chest X-ray dataset. It sequentially loads image metadata and corresponding annotation coordinates from a structured JSON file, mapping specific category IDs to predefined colors (such as red for "Pneumonia" and yellow for "Complications"). By utilizing OpenCV to read the image arrays and Matplotlib to create graphical overlays, the code accurately superimposes bounding box rectangles and text labels directly onto the diagnostic images. This implementation effectively serves as an exploratory data analysis tool, enabling users to efficiently verify the correctness and exact positioning of pathological localizations prior to training.
